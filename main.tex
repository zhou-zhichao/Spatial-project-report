\begin{abstract}
The use of covariance kernels is ubiquitous in the field of spatial statistics. Kernels allow data to be mapped into high-dimensional feature spaces and can thus extend simple linear additive methods to nonlinear methods with higher order interactions. However, until recently, there has been a strong reliance on a limited class of stationary kernels such as the Mat\'ern or squared exponential, limiting the expressiveness of these modelling approaches. Recent machine learning research has focused on spectral representations to model arbitrary stationary kernels and introduced more general representations that include classes of nonstationary kernels. In this paper, we exploit the connections between Fourier feature representations, Gaussian processes and neural networks to generalise previous approaches and develop a simple and efficient framework to learn arbitrarily complex nonstationary kernel functions directly from the data, while taking care to avoid overfitting using state-of-the-art methods from deep learning. We highlight the very broad array of kernel classes that could be created within this framework. We apply this to a time series dataset and a remote sensing problem involving land surface temperature in Eastern Africa. We show that without increasing the computational or storage complexity, nonstationary kernels can be used to improve generalisation performance and provide more interpretable results \citep[p.~59]{ton2018spatial}.
\end{abstract}

\subsection{Gaussian Processes}

A random field is a collection of random variables
\begin{equation}
  X = \{X(\mathbf{s}) : \mathbf{s}\in S\},
\end{equation}
indexed by a set $S$ of locations. It is described through its finite-dimensional distributions, that is, through the distributions of vectors of the form
\begin{equation}
  \left(X(\mathbf{s}_1), \ldots, X(\mathbf{s}_d)\right),
  \qquad \mathbf{s}_1,\ldots,\mathbf{s}_d\in S.
\end{equation}
A Gaussian random field is a random field such that every such finite-dimensional random vector has a multivariate Gaussian distribution. Any Gaussian random field is completely determined by its mean function
\begin{equation}
  m(\mathbf{s}) = \mathbb{E}[X(\mathbf{s})]
\end{equation}
and covariance function
\begin{equation}
  C(\mathbf{s},\mathbf{t}) = \operatorname{Cov}(X(\mathbf{s}),X(\mathbf{t})).
\end{equation}
In this report, Gaussian processes are used in this same finite-dimensional sense. Since the covariance function controls spatial dependence, kernel design is central to Gaussian process models.

\subsection{Stationary and Nonstationary Kernels}

A random field is called weakly stationary if its mean is constant and its covariance depends only on the displacement between two locations:
\begin{equation}
  C(\mathbf{s}, \mathbf{t}) = C_0(\mathbf{t}-\mathbf{s}),
\end{equation}
for some function $C_0$. In the weakly stationary case, one often writes
\begin{equation}
  C(\mathbf{h}) = C_0(\mathbf{h}) = C(\mathbf{s}, \mathbf{s}+\mathbf{h}),
\end{equation}
where $\mathbf{h}$ is the spatial lag. If the covariance depends only on the Euclidean distance $\|\mathbf{h}\|$, the model is also isotropic.

The selected paper points out that spatial modelling has often relied on ``a limited class of stationary kernels such as the Mat\'ern or squared exponential,'' which can limit the expressiveness of the model \citep[p.~59]{ton2018spatial}. In contrast, nonstationary kernels allow the covariance structure to vary with location. The paper therefore considers more general spectral representations and proposes a framework to learn complex nonstationary kernel functions directly from data \citep{ton2018spatial}.

\subsection{Fourier Features}

Bochner's theorem gives the spectral view of stationary covariance modelling. In the course material, its meaning is summarized as follows: every continuous stationary covariance function has a spectral representation; conversely, every such spectral representation defines a valid stationary covariance function. Thus, stationary covariance modelling can be done either in the spatial domain, through covariance decay in space, or in frequency space, through the spectral measure or density.

The selected paper connects this spectral view to Fourier features. It states that recent machine learning research has focused on ``spectral representations to model arbitrary stationary kernels'' and on more general representations including nonstationary kernels \citep[p.~59]{ton2018spatial}. The paper then uses ``the connections between Fourier feature representations, Gaussian processes and neural networks'' to develop a framework that can learn complex nonstationary kernel functions directly from data \citep[p.~59]{ton2018spatial}.
